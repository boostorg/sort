Introduction

This header includes everything in the Boost Algorithm Sorting Library. All of these hybrid algorithms sort quickly in a fashion specialized to the data type being sorted.

sort wraps integer_sort and float_sort, calling whichever is appropriate to the data type, or std::sort if neither is.

integer_sort is a hybrid sort for integers

float_sort is a hybrid sort for floats

string_sort is a hybrid sort for strings

constants are used to tune integer_sort and float_sort. Developers who understand the algorithm in detail may modify them, but otherwise modification is not recommended.
